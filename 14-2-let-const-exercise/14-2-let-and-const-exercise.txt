Quiz
What is the difference between var and let?
Var is able to be redeclared and let is not. Var is function scope and let is block scope.

What is the difference between var and const?
Var is able to be redeclared and reassigned, const is not.

What is the difference between let and const?
Let is able to be redeclared and reassigned, and const is not.

What is hoisting?
Variables are hoisted when they are declared. Variables must be hoisted before we are able to access them.
